Place your provided image files here and name them:

- icon.png    -> the app icon (recommended: 1024x1024 PNG)
- splash.png  -> the splash screen image (recommended: 1242x2436 or similar)

After copying the images to `assets/`, run one of the following to apply them:

1) For fast testing in Expo Go (no native rebuild):
   npx expo start --dev-client

2) To regenerate native resources and build a standalone APK locally (requires Android tooling):
   npx expo prebuild --platform android
   cd android
   .\gradlew.bat assembleRelease

3) To build and get an APK from EAS (recommended):
   npx eas build --platform android --profile <profile-name>

Notes:
- Expo will generate required icon sizes from `assets/icon.png` during prebuild/EAS builds. Use a high-resolution square PNG with transparent background for best results.
- If you want me to place the exact image you uploaded into `assets/icon.png`, download the image from your messages and copy it to this folder, then tell me and I can run the build commands for you.
